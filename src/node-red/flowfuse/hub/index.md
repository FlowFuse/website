# {{ meta.title }}

This section contains documentation for **FlowFuse Certified Nodes** for **Information Technology (IT)**. These nodes connect your FlowFuse instances to enterprise systems, cloud platforms, databases, and business applications.

FlowFuse Certified Nodes are packages that FlowFuse has vetted for quality, security, and support, and maintains on an ongoing basis. To learn more about what certification means and how these nodes are delivered, [read the FlowFuse Certified Nodes blog post](/blog/2025/07/certified-nodes-v2/).

> **ℹ Note:** This section is expanding. We are actively working to bring more IT Certified Nodes to FlowFuse, and additional documentation will be added here over time.

## Accessing the FlowFuse Hub Certified Nodes Catalogue

IT Certified Nodes are available through the **FlowFuse Hub Certified Nodes** catalogue in the FlowFuse editor. To access it:

1. Open the **Palette Manager** from the top-right menu in the editor.
2. Switch to the **Install** tab.
3. Look for the **FlowFuse Hub Certified Nodes** collection.

From there you can browse and install any available IT Certified Nodes directly into your instance.

> **ℹ Note:** Existing devices and hosted instances will not see newly added nodes until they are restarted. Restart any instance you plan to install nodes on so it picks up the updated catalogue.

> **ℹ Note:** The FlowFuse Hub Certified Nodes catalogue is a paid add-on. [Contact us](/contact-us/) to get access or to learn more.
